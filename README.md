## 📚简介
magic-tools 里为hutool的精简版代码，去掉了一些不常用的模块和类


## 📦安装

### 🍊Maven
在项目的pom.xml的dependencies中加入以下内容:

```xml
<dependency>
    <groupId>com.magictools</groupId>
    <artifactId>magic-tools</artifactId>
    <version>1.0.0</version>
</dependency>
```

### 🍐Gradle
```
implementation 'cn.hutool:hutool-all:5.8.4'
```
